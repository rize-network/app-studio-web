# App Studio Web · UI Kit

Interactive recreation of the `@app-studio/web` component library.

## Files

- `index.html` — interactive showcase with navigation, theme toggle, and a mini chat demo
- `components.jsx` — JSX recreations of the core components
  - `Button`, `Card`, `Input`, `TextArea`, `Badge`, `Alert`, `Avatar`, `Checkbox`, `Switch`
  - `Icon` (thin wrapper around Lucide)
  - `ChatMessage` / `ChatInput`
  - `Sidebar`, `TopBar`

## Scope

These are **cosmetic recreations** for prototyping — same visuals, same tokens, ~80% of the real behavior. They are not drop-in replacements for `@app-studio/web` in production.
