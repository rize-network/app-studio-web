# Center

Centers children horizontally and vertically within its bounds

### **Import**
  ```tsx
  import { Center } from '@app-studio/web';
  ```

## Props

| Prop      | Type          | Description                                      | Default |
| --------- | ------------- | ------------------------------------------------ | ------- |
| children  | React.ReactNode | The content to be centered.                       |         |
| styles | CSSProperties    | Optional custom styles for the center container. |         |
