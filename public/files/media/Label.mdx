# Label

Stylable text element supporting various typographic options

### **Import**
  ```tsx
  import { Label } from '@app-studio/web';
  ```

## Props

The `Label` component inherits all props from the [`Text`](./Text.mdx) component.

| Prop      | Type          | Description                                      | Default |
| --------- | ------------- | ------------------------------------------------ | ------- |
| children  | React.ReactNode | The text content of the label.                   |         |
| styles    | CSSProperties | Optional custom styles for the label.            |         |
| className | string        | Optional className for the label.               |         |
