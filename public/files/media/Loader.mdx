# Loader

Provides configurable loading indicators with optional text.

### **Import**
  ```tsx
  import { Loader } from '@app-studio/web';
  ```

## Props

| Prop      | Type          | Description                                            | Default  |
| --------- | ------------- | ------------------------------------------------------ | -------- |
| type      | string        | The type of loader ('spinner', 'dots', 'bar', etc.).    | 'spinner' |
| size      | Size          | The size of the loader ('xs', 'sm', 'md', 'lg', 'xl'). | 'md'     |
| text      | string        | Optional text to display alongside the loader.          |          |
| styles    | CSSProperties | Optional custom styles for the loader.                 |          |
| className | string        | Optional className for the loader container.           |          |
